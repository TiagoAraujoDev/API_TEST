<h3 id="pre-requisitos">Pré-requisitos</h3>

Before starting, you will need to have the following tools installed on your machine:
[NodeJS](https://nodejs.org/en/) and [Git](https://git-scm.com)
Besides, it's good to have an editor to work with the code like [VSCode](https://code.visualstudio.com/)

- Open the terminal in the API directory use the following command lines:
- first install the dependencies with: <code>&lt;npm i&gt;</code>
- and start the server with: <code>&lt;npm start&gt;</code>

<h3 id="objetivo">🎯 Goals</h3>

- Getting better at programming in the only way that works... by practicing
- The major goal of this project was to practice the methods HTTP to consume a REST API made in Node JS.
- I used Bootstrap 5 to syle the app and just a fill adjustments with pure CSS.
